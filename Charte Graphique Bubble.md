## **1\. Identité de Marque**

### **Vision**

Bubble révolutionne l'investissement en démocratisant l'expertise financière grâce à l'IA, avec transparence et efficacité.

### **Personnalité de Marque**

* **Transparente** : Pas de zones d'ombre, tout est explicable  
* **Innovante** : À la pointe de la technologie IA  
* **Accessible** : Finance de pointe pour tous  
* **Éthique** : Redistribution équitable de la valeur

---

## **2\. Logo et Symbole**

### **Logo Principal**

Le logo Bubble se compose de deux éléments distincts :

* **Symbole** : Un cercle ouvert (bulle) avec un point externe  
* **Logotype** : Le texte "Bubble." avec point final

### **Symbolisme**

* **Cercle ouvert** : Représente une bulle ouverte, symbole de transparence et d'accessibilité, en opposition au monde fermé de l'investissement traditionnel réservé à une élite  
* **Point externe (cercle fermé)** : Symbolise l'investissement élitiste et fermé que nous ouvrons grâce à l'IA. Représente aussi l'innovation et l'élément perturbateur  
* **Forme de J** : Le cercle ouvert forme subtilement un "J" en hommage aux prénoms des fondateurs (Joris et Jade)  
* **Point final** : Affirme la finitude et la précision de notre approche

### **Variations du Logo**

#### **Logo Complet Horizontal**

\[Symbole\] Bubble.

#### **Logo Seul (Symbole)**

Cercle ouvert avec point externe \- utilisable indépendamment **Usage** : Quand l'espace est limité ou en format vertical sans place pour le texte

#### **Logo Vertical**

Quand l'espace horizontal est insuffisant :

* **Si espace suffisant** : Utiliser le logo complet vertical avec "Bubble."  
* **Si espace limité** : Utiliser uniquement le symbole sans texte

### **Versions du Logo**

#### **Logo sur Fond Blanc**

* **Symbole** : Noir `#000000`  
* **Texte** : Noir `#000000`  
* **Usage** : Support principal recommandé pour tous les formats

#### **Logo sur Fond Noir**

* **Symbole** : Blanc `#FFFFFF`  
* **Texte** : Blanc `#FFFFFF`  
* **Usage** : Uniquement quand le contexte l'exige (fonds sombres obligatoires)

### **Spécifications Techniques**

#### **Proportions**

* **Espacement symbole/texte** : 0.18em  
* **Ratio symbole/texte** : Le symbole fait environ 1.5x la hauteur du texte  
* **Point final** : Partie intégrante du logo, jamais omis

#### **Tailles Minimales**

* **Web** : 120px de largeur minimum  
* **Print** : 25mm de largeur minimum  
* **Favicon** : 72x72px (version simplifiée)

## **3\. Couleurs**

### **Palette Principale**

#### **Noir Principal**

* **Hex** : `#000000`  
* **RGB** : `0, 0, 0`  
* **Usage** : Logo, textes principaux, contours

#### **Gris Foncé (Primary)**

* **Hex** : `#333333`  
* **RGB** : `51, 51, 51`  
* **Usage** : Titres, boutons principaux, éléments interactifs

#### **Gris Foncé Hover**

* **Hex** : `#444444`  
* **RGB** : `68, 68, 68`  
* **Usage** : États de survol des éléments interactifs

### **Palette Secondaire**

#### **Blanc (Arrière-plan Privilégié)**

* **Hex** : `#FFFFFF`  
* **RGB** : `255, 255, 255`  
* **Usage** : **Arrière-plan principal recommandé pour tous supports** \- reflète la transparence et la simplicité Bubble

#### **Gris Clair (Card)**

* **Hex** : `#F8F8F8`  
* **RGB** : `248, 248, 248`  
* **Usage** : Arrière-plans de cartes, sections alternées

#### **Gris Très Clair (Borders)**

* **Hex** : `#EEEEEE`  
* **RGB** : `238, 238, 238`  
* **Usage** : Bordures, séparateurs

#### **Gris Moyen (Texte secondaire)**

* **Hex** : `#666666`  
* **RGB** : `102, 102, 102`  
* **Usage** : Textes secondaires, descriptions

### **Règles d'Usage**

* **Contraste minimum** : Toujours respecter un ratio de 4.5:1 pour l'accessibilité  
* **Dominante** : Le blanc et les gris clairs dominent pour une sensation de transparence  
* **Accents** : Le noir et gris foncé pour les éléments importants

## **4\. Typographie**

### **Police Principale : Inter**

* **Famille** : Inter (Google Fonts)  
* **Fallback** : sans-serif  
* **Caractéristiques** : Moderne, lisible, optimisée pour les écrans

### **Hiérarchie Typographique**

#### **Titre Principal (H1)**

* **Taille** : 3.5rem (mobile) / 4rem (desktop)  
* **Poids** : 800 (Extra Bold)  
* **Espacement** : \-0.03em  
* **Usage** : Titre hero uniquement

#### **Titres Sections (H2)**

* **Taille** : clamp(1.7rem, 4vw, 2.5rem)  
* **Poids** : 700 (Bold)  
* **Espacement** : \-0.02em  
* **Couleur** : var(--primary) \#333333

#### **Sous-titres (H3)**

* **Taille** : 1.75rem  
* **Poids** : 700 (Bold)  
* **Usage** : Titres de cartes principales

#### **Titres Secondaires (H4)**

* **Taille** : clamp(1.15rem, 2.5vw, 1.5rem)  
* **Poids** : 700 (Bold)  
* **Usage** : Titres de features

#### **Corps de Texte**

* **Taille** : 1rem (base) / 1.1rem (contenu important)  
* **Poids** : 400 (Regular)  
* **Interligne** : 1.6 à 1.8  
* **Couleur** : \#444444

#### **Texte Tagline (Variable)**

* **Taille** : 1.5rem  
* **Poids** : 500 (Medium)  
* **Style** : **Toujours en italique**  
* **Usage** : Phrases d'accroche sous le logo (contenu variable selon le contexte)  
* **Exemple** : *"L'investissement optimisé à l'ère de l'IA"*, *"Rendre l'investissement transparent, moderne et accessible"*

### **Style d'Icônes**

* **Type** : Icônes au trait (line icons)  
* **Épaisseur** : 1.5 à 2px  
* **Style** : Minimaliste, géométrique  
* **Source recommandée** : Lucide React, Heroicons

* **Arrière-plan** : `rgba(255, 255, 255, 0.7)`  
* **Bordure** : `rgba(255, 255, 255, 0.3)`  
* **Backdrop-filter** : `blur(10px)`  
* **Usage** : Sections de chat, cartes flottantes

#### **Ombres**

* **Légère** : `0 4px 6px rgba(0, 0, 0, 0.05)`  
* **Modérée** : `0 8px 24px rgba(0, 0, 0, 0.04)`  
* **Forte** : `0 16px 32px rgba(0, 0, 0, 0.08)`

#### **Rayons de Bordure**

* **Standard** : 8px à 16px  
* **Cartes importantes** : 16px à 20px  
* **Boutons** : 8px à 12px

## **7\. Animations et Transitions**

### **Principes**

* **Durée** : 0.3s à 0.6s  
* **Timing** : `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out naturel)  
* **Style** : **Subtiles, efficientes et futuristes** \- créer un climat tech en accord avec l'identité de marque  
* **Philosophie** : Mouvements qui renforcent l'impression de modernité technologique sans distraire de l'expérience utilisateur

### **Animations Standard**

* **Hover cards** : `translateY(-5px)` \+ ombre renforcée  
* **Fade-in** : Opacité 0→1 \+ `translateY(20px→0)`  
* **Boutons** : Scale léger et changement d'ombre

### **Animation du Logo (Recommandée)**

* **Animation au survol** : Le cercle se dessine dans le sens des aiguilles d'une montre, puis le point vient se déposer comme pour terminer un "J"  
* **Durée totale** : 0.8s à 1.2s  
* **Séquence** :  
  1. Cercle : `stroke-dashoffset` animation sur 0.6s  
  2. Point : Apparition avec `scale(0→1)` \+ `translateY(-10px→0)` sur 0.3s (démarrage à 0.5s)  
* **Timing** : `cubic-bezier(0.25, 0.1, 0.25, 1)` pour un effet naturel  
* **Usage** : Logo principal sur site web, éléments interactifs

### **Animation de Bulle (Éléments UI)**

* **Float effect** : Mouvement vertical subtil pour les éléments circulaires  
* **Glassmorphisme** : Effets de transparence et de flou

## **11\. Usage et Guidelines**

### **À Faire ✅**

* **Privilégier systématiquement les fonds blancs** et le logo noir sur blanc  
* Utiliser le logo dans ses proportions originales  
* Respecter l'espace minimal autour du logo  
* **En format vertical contraint : utiliser uniquement le symbole** (pas de texte "Bubble.")  
* Maintenir la lisibilité sur tous les supports  
* **Toujours mettre les taglines en italique** (contenu variable selon le contexte)  
* Respecter la hiérarchie typographique  
* Créer des animations subtiles et futuristes

### **À Éviter ❌**

* Modifier les proportions du logo  
* Omettre le point final de "Bubble."  
* **Écrire "Bubble." en format vertical si l'espace est insuffisant** (utiliser uniquement le symbole)  
* Utiliser d'autres polices que Inter  
* Créer des variantes de couleur non autorisées  
* **Utiliser des fonds sombres sauf contrainte technique absolue**  
* **Omettre l'italique sur les taglines**  
* Placer le logo sur des fonds à faible contraste  
* Créer des animations lourdes ou distrayantes
